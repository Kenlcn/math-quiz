// 🎯 智慧比對多項式解析器 (終極修復版：精準分離 x2、x 與常數項) function parsePolynomial(str, targetT2, targetT3) { let clean = str.toLowerCase().replace(/\s+/g, '').replace(/[()=^]/g, ''); // 基礎手寫誤差轉換 clean = clean.replace(/[loij]/g, '1'); clean = clean.replace(/[zq]/g, '2'); clean = clean.replace(/[s]/g, '5'); clean = clean.replace(/[b]/g, '6'); clean = clean.replace(/[g]/g, '9'); // 建立完美答案比對鏡 let perfectAns = "x2"; if (targetT2 > 0) perfectAns += "+" + targetT2 + "x"; else if (targetT2 < 0) perfectAns += targetT2 + "x"; if (targetT3 > 0) perfectAns += "+" + targetT3; else if (targetT3 < 0) perfectAns += targetT3; let perfectAnsShort = perfectAns; if (targetT2 === 1) perfectAnsShort = perfectAns.replace("1x", "x"); if (targetT2 === -1) perfectAnsShort = perfectAns.replace("-1x", "-x"); if (clean === perfectAns || clean === perfectAnsShort) { return { x2Coeff: 1, xCoeff: targetT2, constValue: targetT3, hasX2: true, hasX: (targetT2 !== 0), hasConst: (targetT3 !== 0) }; } // 強健拆項邏輯：先將 x2 這種特殊特徵保護起來，再透過正則拆分項 let x2Coeff = 0, xCoeff = 0, constValue = 0; let hasX2 = false, hasX = false, hasConst = false; // 用正則表達式精準切出帶有正負號的各個多項式項 let rawTerms = clean.split(/(?=[+-])/).filter(t => t !== ""); rawTerms.forEach(t => { // 判斷是否為 x^2 項 (包含 x2, x^2) if (t.includes('x2') || t.includes('x^2')) { hasX2 = true; let coeffStr = t.replace(/x2|x\^2/g, ''); if (coeffStr === "" || coeffStr === "+") x2Coeff = 1; else if (coeffStr === "-") x2Coeff = -1; else { let p = parseInt(coeffStr); x2Coeff = isNaN(p) ? 1 : p; } } // 判斷是否為一次項 (有 x 但不是 x2) else if (t.includes('x')) { hasX = true; let coeffStr = t.replace(/x/g, ''); if (coeffStr === "" || coeffStr === "+") xCoeff = 1; else if (coeffStr === "-") xCoeff = -1; else { let p = parseInt(coeffStr); xCoeff = isNaN(p) ? 0 : p; } } // 純常數項 else { let v = parseInt(t); if (!isNaN(v)) { hasConst = true; constValue = v; } } }); return { x2Coeff, xCoeff, constValue, hasX2, hasX, hasConst }; }
